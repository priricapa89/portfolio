# PRicapa.ai

Personal website for Pricilla Ricapa — AI & Quantum Strategist, Educator, and Speaker.

## About

A single-page personal brand site built with vanilla HTML, CSS, and JavaScript. Features a responsive layout with sections for biography, gallery, speaking topics, services, events, and contact.

## Structure

- `index.html` — main site (hero, about, gallery teaser, book teaser, topics, Humanist SI, services, events, contact)
- `gallery.html` — full photo diary of events (WEF Davos, TEDx, MIT, Harvard Quantum, Dassault Systèmes, etc.)

## Tech

- Vanilla HTML/CSS/JS — no frameworks or build tools
- Fonts: Playfair Display, DM Sans, DM Mono (Google Fonts)
- Fully responsive: tablet (≤960px) and mobile (≤600px) breakpoints
- Cloudflare email obfuscation

## Deployment

Hosted as a static site. Push to `origin/main` to deploy.
